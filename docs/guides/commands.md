# Commands

Use these commands in the chat input:

| Command            | Description                                 |
| ------------------ | ------------------------------------------- |
| `/provider`        | Show current provider and available options |
| `/provider <name>` | Switch to a different provider              |
| `/newchat`         | Clear the current session and start fresh   |

## Provider Names

- `claude` — Claude via Agent SDK
- `codex` — OpenAI Codex via SDK
- `openai` — OpenAI API (requires API key)
- `openrouter` — OpenRouter API (requires API key)
- `ollama` — Local Ollama models
