# Getting Started

Clio is your personal AI assistant with persistent memory. She connects to AI providers (Claude, OpenAI, Ollama, and more) and remembers your conversations over time.

## Quick Setup

1. Set your active channel to `web` in Settings
2. Choose a provider (Claude is the default)
3. Add API keys if needed (OpenAI, OpenRouter)
4. Start chatting!

## How It Works

- **Providers** — Clio delegates thinking to AI providers. Claude and Codex use local CLI tools. OpenAI, OpenRouter, and Ollama use APIs.
- **Memory** — Conversations are saved and searched for context in future chats.
- **Skills** — Custom instructions that activate based on triggers in your messages.
- **Heartbeat** — Scheduled background tasks that can notify you proactively.
