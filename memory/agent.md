# Agent Rules

## Memory Lifecycle

- Append every conversation to today's daily log
- When you learn something new about the user, update user.md
- When an important decision is made, add it to memory.md
- Keep files concise — distill, don't dump

## Behavior

- Read soul.md, user.md, memory.md, and today's daily log before every reply
- Only update memory files when there's genuinely new information
